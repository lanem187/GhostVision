OVERVIEW
RF Ghost Vision transforms your browser into a cyberpunk RF analysis 
station. Simulates Wi-Fi 6E CSI motion detection, BLE proximity 
tracking, and UWB ranging to reconstruct human presences as pulsing 
ghost silhouettes in AR overlay. Four modes: GHOST, RADAR, SCAN, 
TRACKING. Pure React + SVG. Zero dependencies beyond React 18.

CORE CAPABILITIES
├── SIGNAL SCANNING ENGINE
│   ├── Wi-Fi 6E (2.4/5/6GHz) • 50+ realistic SSIDs • MAC OUI lookup
│   ├── BLE devices (wearables/audio/trackers) • Connectable status
│   ├── UWB ranging • Azimuth/elevation • LOS detection
│   └── CSI variance → motion scoring (0.0-1.0)
│
├── SENSOR FUSION
│   ├── IMU simulation (accelerometer/gyro/compass)
│   ├── World→screen transforms (FOV 60°)
│   └── Phone position tracking (x,y,z meters)
│
├── AI PRESENCE ENGINE
│   ├── Multi-signal fusion (Wi-Fi+BLE+UWB)
│   ├── Silhouette generation (height/motion/velocity)
│   ├── Confidence scoring (30-98%)
│   └── Nearby detection merging (±1.5m threshold)
│
├── VISUALIZATION MODES
│   ├── GHOST: AR camera overlay with pulsing silhouettes
│   ├── RADAR: 360° sweep with entity pings + heatmap
│   ├── SCAN: Device tables with RSSI meters
│   └── TRACKING: Signal intel + security analysis
│
└── EXPORT: JSON logs (entities, signals, timestamps)
